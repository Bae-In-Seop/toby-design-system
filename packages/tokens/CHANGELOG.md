# @toby-design/tokens

## 1.1.0

### Minor Changes

- Add semantic token architecture with dark theme support. Tokens are now structured in primitive/light/dark layers. Components use new semantic token names (--color-surface, --color-on-primary, --color-border). Added ThemeProvider and useTheme hook for React-native theme control.

## 1.0.0

### Major Changes

- Add design token pipeline with 49 tokens across 9 categories (color, spacing, radius, fontSize, fontWeight, lineHeight, shadow, transition, zIndex)
