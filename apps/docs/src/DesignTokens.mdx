# Design Tokens

이 디자인 시스템은 **Primitive 토큰**과 **Semantic 토큰** 두 계층으로 구성됩니다.
토큰을 변경하면 모든 컴포넌트에 자동 반영됩니다.

> 상단 툴바의 **Theme** 버튼으로 Light / Dark 모드를 전환하며 토큰 변화를 확인할 수 있습니다.

---

## Token Architecture

```
┌─────────────────────────────────────────────┐
│  Primitive Tokens (테마 무관)                  │
│  spacing, radius, fontSize, fontWeight, ...  │
│  → :root 에 항상 고정                          │
├─────────────────────────────────────────────┤
│  Semantic Tokens (테마별 다름)                  │
│  color-surface, color-primary, shadow, ...   │
│  → Light: :root (기본)                        │
│  → Dark:  [data-theme="dark"] (오버라이드)     │
└─────────────────────────────────────────────┘
```

**Primitive 토큰**은 spacing, radius처럼 테마와 무관한 고정값입니다.
**Semantic 토큰**은 color, shadow처럼 Light/Dark 테마에 따라 값이 달라지는 토큰입니다.

## Dark Theme

시맨틱 토큰은 `[data-theme="dark"]` 선택자로 오버라이드됩니다.
컴포넌트는 CSS 변수명만 참조하기 때문에 별도의 코드 변경 없이 테마가 전환됩니다.

```css
/* Light (기본) */
:root {
  --color-surface: #FFFFFF;
  --color-text: #111827;
  --color-primary: #2563EB;
  --color-border: #E5E7EB;
}

/* Dark (오버라이드) */
[data-theme="dark"] {
  --color-surface: #0f172a;
  --color-text: #f1f5f9;
  --color-primary: #3b82f6;
  --color-border: #334155;
}
```
